# Extension Ideas

Ideas for extending beyond the base AXIOM replication.

## Potential Directions

- **Different environments**: Test AXIOM on environments beyond Gameworld 10k
- **Learned projections**: Replace fixed A/B projection matrices in sMM with learned ones
- **Hierarchical dynamics**: Add a second level of temporal abstraction above tMM
- **Transfer learning**: Pre-train on one game, fine-tune on another
- **Scalability**: Profile and optimize the inference loop for larger state spaces
- **Alternative planning**: Compare expected free energy with other planning objectives
- **Richer object interactions**: Extend rMM to handle multi-object (not just pairwise) interactions
